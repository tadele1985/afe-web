# AFE Web

## Run project
1. Activate the virtual environment using [poetry](https://python-poetry.org/)
```bash 
poetry shell
```
2. Install tailwind and daisyui
``` bash
pnpm install
```
3. Start tailwind to watch for changes in your files and reload the assets.
```bash
pnpm dev
```
4. Run the django server
```bash
python manage.py runserver
```
